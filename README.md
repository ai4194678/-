# -
Этот проект представляет собой мини-исследование на основе данных 250,000 студентов. Основная цель — выявить взаимосвязь между образом жизни (сном, учебой) и психологическим состоянием (стрессом, выгоранием). В ходе работы были проанализированы ключевые метрики благополучия и их влияние на академическую успеваемость.
